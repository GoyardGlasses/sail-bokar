================================================================================
                      TROUBLESHOOTING
================================================================================

PROBLEM: White screen on startup
SOLUTION: Wait 10-15 seconds. The dev server needs time to start.

PROBLEM: Port already in use
SOLUTION: The system will automatically find the next available port.
          Check the terminal for the actual URL.

PROBLEM: Backend not starting
SOLUTION: Make sure Python is installed:
          python --version

PROBLEM: Dependencies missing
SOLUTION: Run REBUILD-SYSTEM.bat to reinstall everything.

PROBLEM: Still having issues?
SOLUTION: 
  1. Close all terminals and the app
  2. Run VERIFY-SYSTEM.bat to check prerequisites
  3. Run REBUILD-SYSTEM.bat to rebuild everything
  4. Try START-APP.bat again
